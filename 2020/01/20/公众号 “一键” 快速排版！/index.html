公众号“一键”快速排版！

2020-01-20
其他
字数统计: 2.3k字   |   阅读时长≈ 10分钟

公众号排版一直是一个耗时费力的事情，市面上存在各大排版平台，虽然自由度高，但对于板式相对固定，大部分只是更换图文的排版来说，排版平台过于繁琐，浪费时间。本文为分享我是如何快速排版我的文章❤

markdown 介绍

markdown 语法，可以利用简单标记语法，使普通文本具有格式，支持代码，脚注，表格，图片，各类形式的文本，同时可以导出成 latex，word，pdf 等进行后续的调整，同时，可以编写 css 语法生成特定格式图文详细，上手容易，排版方便。

|markdown 语法一览

标题样式

  • H1 :# Header 1

  • H2 :## Header 2

  • H3 :### Header 3

  • H4 :#### Header 4

  • H5 :##### Header 5

  • H6 :###### Header 6

文本样式

  • 链接 : [文 字][链接内容]

  • 加粗 :** 加粗 **

  • 斜体字 :* 倾斜

  • 段落 : 段落之间空一行

  • 列表 :* 添加星号成为一个新的列表项。

  • 有序列表： 1. 加上空格

  • 引用 :> 引用内容

  • 代码 : ```

  • 画线 :——–

|markdown 缺点

markdown 纯文本的写作同样存在一个弊端，在于不能粘贴图片，如果你是网络上查找的图片，插入图片时，只需要嵌入图片网址就行；如果你是本地的图片，markdown 嵌入图片，嵌入的是本地的地址，不像 word 嵌入图片后删掉本地图片不会影响图片在 word 中的视图，所以对于 Markdown 而言图床的搭建是一种简单方便的解决方法。

| 图床

主流的图床工具有 SM.MS，腾讯云，微博，Github, 七牛云，Imgur，阿里云 OSS 等平台，有免费的也存在付费的，目前该文在使用的图床是阿里云 OSS。

同时，搭配使用的有 PicGo 图床管理工具，PicGo 可以方便上传图片，转为 markdown 语法链接，可视化程度比较高。PicGo 支持多系统端口，多图床平台。（下载地址： https://github.com/Molunerfinn/PicGo/releases ）

根据自己系统选择下载对应版本，下载完成，以我为例，找到阿里图床 id 以及 keysecret，添加阿里 oss 设置。

配置完成后，就可以上传管理图片，支持剪贴板截图上传，在 markdown 编辑器直接粘贴即可。

|markdown 编辑器

最后推荐下 markdown 编辑器 typora[https://www.typora.io/] ，支持多平台，可以粘贴图片，表格。mac 端可以直接上传图床。导出格式兼容性较好，支持大纲 / 文件视图可以自主切换代码格式，和书写视图的模式，专注模式下更容易专注于文本内容。

利用 typora 完成图文撰写后，接下来我们到排版部分。

一键自动排版

排版推荐使用 Md2All，将完成的 mardown 文本以 markdown 格式复制，粘贴到 Md2All[http://md.aclickall.com/] 网站

左侧为 mardown 代码区域，右侧为预览效果区域，在一键排版选项下，可以选择网站附带的几种样式，同时如果对 css 代码有一定了解，可以自己修改网站上的代码纯为自己公众号的模板。

Md2All 的代码注释比较全面，只要根据注释的内容进行调整即可。调整完模板可以将代码复制保存以备下次使用。（结尾附上我自己调整过的 css 代码）

最后，复制文本到公众号编辑界面粘贴即可。

注 ：
css 同时也可以作为 typora（或者 mweb 等编辑器）主题，在 Md2All 调试完成可以在剪辑器里面直接导入，导出为 html 格式复制同样可以。
